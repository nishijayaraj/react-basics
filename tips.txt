#1. To create a project using Vite
     npm create vite@latest
This would ask for project name, frameworks and a lot of other info. For more details refer to https://flaviocopes.com/vite-react-app/

#2. starting this app 
    npm run dev  ( script tag in package.json => npm run vite)
